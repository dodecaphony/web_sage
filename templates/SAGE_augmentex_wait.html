Augmentation: Augmentex

An augmentation based on heuristics is available on this page. To use the augmentation, you need to specify the language of the original text and the level at which you want the augmentation to be performed. There are two options: word level or symbol level. You also need to provide some parameters, such as the phrase to which augmentations will be applied, the minimum and maximum number of augmentations, and a random seed for reproducibility. You can either enter the text directly in a special field or upload a file in a supported format. Data can also be augmented using distributions (SBSC) or a pipeline on the relevant pages.
