Augmentation: SBSC

On this page, you can use a tool to correct erroneous sentences. To do so, you need to choose the language of the original text. You can enter the text in a special field, or upload a file in a supported format. After that, you need to select the dataset based on which the error distribution will be calculated. Data can also be augmented using heuristics (Augmentex) or a pipeline on the relevant pages.
