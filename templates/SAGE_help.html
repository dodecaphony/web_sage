About SAGE

Key features

  • Multilingual spelling correction with support for Russian and English
  • Various models of different sizes and characteristics to optimize speed and quality
  • Augmentation methods for generating training data
  • Web interface with a convenient input form and file download capabilities
  • Asynchronous text processing without blocking the interface
  • Support for different file formats (TXT, CSV, TSV)
  • REST API for easy integration with other systems
  • Results in CSV format for easy export

    • Input data

    The system supports the following file formats:

  • TXT - text files
  • CSV - comma separated values files
  • TSV - tab separated values files

    • Asynchronicity

    SAGE uses asynchronous task processing for working with large texts:

    1. After sending a request, a unique `task_id` is generated
    2. The user is redirected to a waiting page
    3. Every 10 seconds, the page automatically refreshes
    4. When processing completes, a redirect to the results page occurs

    Usage

    Spelling correction
    1. Choose the language of the text
    2. Write or upload the text (TXT, CSV, TSV)
    3. (Optional) Allow data collection for analysis
    4. Receive the corrected text and download the results
    Augmentation with Augmentex
    1. Choose the level of augmentation (chars or words)
    2. Select the language
    3. Configure settings:
      1. Specify the percentage of the phrase to be augmented (unit_prod)
      2. Set the minimum and maximum number of changes
      3. Enter a random seed number
    4. Enter your text or upload a file
    5. Receive a generated text
    Augmentation with SBSC
    1. Choose the language of the text
    2. Enter your text or upload a file
    3. Choose the dataset for calculating the error distribution
    4. Receive the augmented text
    Augmentation with Pipeline
    1. Select a sequence of augmentation methods (up to three)
    2. Configure settings and specify the language
    3. Enter your text or upload a file
    4. Choose the dataset (for SBSC method)
    5. Receive the augmented text
